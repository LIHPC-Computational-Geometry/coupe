Struct coupe::Real

source · 
pub struct Real(/* private fields */);
Expand description

A floating-point value that cannot be NAN nor infinity.

It implements every trait f64 implements, plus std::cmp::Ord and std::cmp::Eq.

Example

let a = Real::from(7.0);
let b = Real::from(5.0);
println!("{} / {} = {}", a, b, a / b);
