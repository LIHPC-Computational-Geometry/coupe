Expand description

Greedy number partitioning algorithm

Greedily assign weights to each part.

Example

use coupe::Partition as _;

let weights = [3.2, 6.8, 10.0, 7.5];
let mut partition = [0; 4];

coupe::Greedy { part_count: 2 }
    .partition(&mut partition, weights)?;

Reference

Horowitz, Ellis and Sahni, Sartaj, 1974. Computing partitions with applications to the knapsack problem. J. ACM, 21(2):277–292. doi:10.1145/321812.321823.
